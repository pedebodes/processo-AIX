# README #

Sistema de controle de alunos, sistema de teste de processo seletivo para a AIX Sistemas

### Requisitos ###
Linguagem backend: 
	PHP 5.6
Framework:
	Laravel 5.4
	ViaCep
Base de dados:
	Mysql
Web:
	AdiminLte
	Jquery


### Hospedagem ###
O sistema foi hospedado, no ambiente gratuito Heroku (https://www.heroku.com/), 
utilizando o  gerenciador de container Docker, na seguinte URL: http://processo-aix.herokuapp.com/


### Configuração da aplicação ###
Configurar o arquivo .env com os dados de cobnexão do banco de dados.


#Base de Testes#
O Sistema esta sendo disponibilizado com o banco de dados completamente vazio, para que o mesmo possa ser populado no teste de funcionalidade da aplicação
